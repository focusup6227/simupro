# **App Name**: EMS Simu-Pro

## Core Features:

- User Authentication and Role Selection: Allow users to create accounts and select their EMS role (EMT, AEMT, Paramedic, Student).
- Scenario Library: Display a library of EMS simulation scenarios for different patient presentations.
- AI-Powered Patient Simulation: Present patient simulations where the AI tool provides realistic patient responses, vital signs, and dynamic condition changes based on user's assessment and treatment choices.
- Assessment Input: Allow users to input patient assessment findings (e.g., vital signs, physical exam).
- Treatment Selection: Enable users to select treatments based on their assessment findings, adhering to national EMS standards.
- Performance Metrics: Track and display user performance metrics such as assessment accuracy, treatment appropriateness, and time to intervention.
- Admin Panel: Provide an admin interface to create/edit scenarios, manage users, and access performance data. (Admin Only)

## Style Guidelines:

- Primary color: Deep Blue (#2E5CB8) for trustworthiness and competence.
- Background color: Light Gray (#E9ECEF), provides a professional and calming feel.
- Accent color: Orange (#D47732), use for alerts, key actions and highlights, analogous to the blue.
- Body and headline font: 'Inter', a grotesque sans-serif with a modern, neutral, machined look.
- Use clear, professional medical icons.
- Clean, intuitive layout with clear sections for scenarios, assessment input, and performance metrics.
- Subtle animations for feedback, alerts, and key actions.
